```

C -> 3
...
Z -> 26

AB -> 28 
...
```

```
示例 1：
输入：columnNumber = 1
输出："A"

示例 2：

输出："AB"

示例 3：
输入：columnNumber = 701
输出："ZY"

示例 4：
输入：columnNumber = 2147483647
输出："FXSHRXW"
```

提示：

1 <= columnNumber <= 2^31 - 1
