148.排序链表

2021-04-21
leetcode linkedlist

题目

给你链表的头结点 head ，请将其按 升序 排列并返回 排序后的链表 。

进阶： 你可以在 O(n log n) 时间复杂度和常数级空间复杂度下，对链表进行排序吗？  

示例 1：
4 -> 2 -> 1 -> 3
排序后：
1 -> 2 -> 3 -> 4

输入：head = [4,2,1,3]
输出：[1,2,3,4]

示例 2：
-1 -> 5 -> 3 -> 4 -> 0
排序后：
1 -> 0 -> 3 -> 4 -> 5

输入：head = [-1,5,3,4,0]
输出：[-1,0,3,4,5]

示例 3：
输入：head = []
输出：[]

提示：

  • 链表中节点的数目在范围 [0, 5 * 10^4] 内
  • -10^5 <= Node.val <= 10^5
