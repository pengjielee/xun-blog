## 安装

```
pip install requests
```

## 导入

```
import requests
```

## 发送get请求

```
r = requests.get('http://www.baidu.com')
r.status_code #200
r.text
r.encoding
r.cookies
r.headers
r.headers['content-type']
```

## 发送put请求（自定义头部）

