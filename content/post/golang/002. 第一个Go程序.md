## 第一个Go程序，打印Hello World

创建文件`hello.go`，内容如下：

```go
// package main表示一个可独立执行的程序，每个 Go 应用程序都包含一个名为 main 的包。
package main

//导入 fmt 包
import "fmt"

// 这是单行注释

/* 
	这是多行注释
	这是多行注释
*/

//程序开始执行的函数
func main() {
	/* 打印Hello World， 并换行 */
	fmt.Println("Hello World")
}
```


## 格式化代码

```
$ go fmt hello.go
```

## 执行Go程序

```bash
$ go run hello.go
Hello World
```

## 生成二进制文件

```bash
$ go build hello.go
$ ls
hello		hello.go
$ ./hello
Hello World
```
