## 判断用户年龄

```go
package main

import (
	"fmt"
)

func main() {
	var age int

	fmt.Printf("请输入年龄：")
	fmt.Scanln(&age)

	if age >= 18 {
		fmt.Println("您已成年，可以上网。")
	} else {
		fmt.Println("未成人不能上网！")
	}
}
```
