## 接受用户名问好

```Python
name = input('Please enter your name: ')
print('Hello,' + name + '!')
```

## 接受一个数值

```Python
age = input('How old are you? ')

if (age.isdigit()):
  if (int(age) > 18):
    print("You're not a kid anymore.")
  else:
    print("You're just a kid.")
else:
  print('Please enter a number.')
```
