## 创建类

创建 animal.py 文件，定义动物类。

```Python
class Animal:
  """初始化方法"""
  def __init__(self, name, age):
    self.name = name
    self.age = age

  """方法eat"""
  def eat(self):
    print(self.name.title() + ' ' + 'is eating.')

  """方法run"""
  def run(self):
    print(self.name.title() + ' ' + 'is running.')
```

## 创建实例

创建动物类的实例。

```Python
dog = Animal('xiaohua', 6)
print(dog.name) # 访问属性name
print(dog.age)  # 访问属性age
dog.eat()       # 调用方法eat()
dog.run()       # 调用方法run()

# output: 
"""
xiaohua
6
Xiaohua is eating.
Xiaohua is running.
"""
```

## 继承

创建 dog.py 文件，定义 Dog 类，Dog 类 继承 Animal类。

```Python
from animal import Animal

class Dog(Animal):
  """初始化方法"""
  def __init__(self, name, age, color):
    super().__init__(name, age)
    self.color = color

  """方法eat"""
  def eat(self):
    print(self.name.title() + ' ' + 'is eating.')

  """重新方法run"""
  def run(self):
    print(self.name.title() + ' ' + "is running. It's color is " + self.color + '.')

  """定义新方法bark"""
  def bark(self):
    print(self.name.title() + '' + 'Wang Wang Wang.')
```

创建实例

```Python
xiaohua = Dog('xiaohua',6,'white')
print(xiaohua.name)
print(xiaohua.age)
print(xiaohua.color)
xiaohua.run()
xiaohua.bark()

# output
"""
xiaohua
6
white
Xiaohua is running. It's color is white.
Xiaohua Wang Wang Wang.
"""
```
