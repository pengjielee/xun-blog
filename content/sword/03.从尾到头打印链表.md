## 题目

输入一个链表，按链表从尾到头的顺序返回一个ArrayList。

## 思路

利用JS数组的unshift()方法，在数组开头插入数据。

```
var nums = [1,2,3];
var temp = nums.unshift(4);
console.log(temp); //4
console.log(nums); //[4,1,2,3]

var nums = [1,2,3];
var temp = nums.shift(); 
console.log(temp); //1
console.log(nums); //[2,3]
```

## JS实现

```javascript
/*function ListNode(x){

    this.next = null;
}*/
function printListFromTailToHead(head) {
  // write code here
  let result = [];

  if (head === null) {
    return result;
  }

  let current = head;
  while (current != null) {
    result.unshift(current.val);
    current = current.next;
  }
  return result;
}
```
