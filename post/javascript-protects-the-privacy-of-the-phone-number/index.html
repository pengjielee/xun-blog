JavaScript保护手机号隐私(中间4位用 * 号替换)

2021-02-08
javascript

需求：将手机号码中间4位用 * 号替换，保护隐私。

使用正则

const encryptPhoneNumber = (str) => {
  if (str) {
    str = str.trim();
    const reg = /(\d{3})\d{4}(\d{3})/;
    str = str.replace(reg, "$1****$2");
  }
  return str;
};

encryptPhoneNumber("18614023235"); //output: 186****3235
