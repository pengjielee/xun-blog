922.按奇偶排序数组II

2021-04-17
leetcode array

题目

给定一个非负整数数组 A， A 中一半整数是奇数，一半整数是偶数。

对数组进行排序，以便当 A[i] 为奇数时，i 也是奇数；当 A[i] 为偶数时， i 也是偶数。

你可以返回任何满足上述条件的数组作为答案。

示例：

输入：[4,2,5,7]
输出：[4,5,2,7]
解释：[4,7,2,5]，[2,5,4,7]，[2,7,4,5] 也会被接受。

提示：

  • 2 <= A.length <= 20000
  • A.length % 2 == 0
  • 0 <= A[i] <= 1000

JS实现

/**
 * @param {number[]} nums
 * @return {number[]}
 */
var sortArrayByParityII = function (nums) {
  const result = new Array(nums.length);
  let i = 0;

  // 第1次遍历，在偶数位上设置偶数
  for (const num of nums) {
    if (num % 2 === 0) {
      result[i] = num;
      i += 2;
    }
  }

  i = 1;
  // 第2次遍历，在奇数位上设置奇数
  for (const num of nums) {
    if (num % 2 === 1) {
      result[i] = num;
      i += 2;
    }
  }
  return result;
};
