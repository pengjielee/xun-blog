使用 npm scripts 格式化

package.json

{
  scripts: {
    "prettier": "prettier --write src/*.js src/**/*.js",
    "prettier:check": "prettier --check src/*.js src/**/*.js"
  }
}

prettier –check，用于检查哪些文件需要格式化；
prettier –write，用于格式化代码；
