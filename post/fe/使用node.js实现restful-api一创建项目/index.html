0001-01-01

1、创建项目目录

mkdir my-expressjs-api && cd my-expressjs-api

2、初始化package.json文件

npm init -y

3、安装express

npm install express --save

4、创建src/server.js

const express = require("express");
const app = express();
const port = 3000;

app.get("/", (req, res) => {
  res.send("Hello World!");
});

app.listen(port, () => {
  console.log(`My app listening at http://localhost:${port}`);
});

5、更新package.json

{
  ...
  "scripts": {
    "dev": "node src/server.js"
  },
  ...
}

6、运行项目

npm run dev

输出：

> node src/server.js

My app listening at http://localhost:3000

访问浏览器http://localhost:3000，页面返回 Hello World!
