Nextjs配置antd语言包

2021-06-22
nextjs

需求

antd 目前的默认文案是英文，我们需要使用中文。

nextjs自定义 App

创建./pages/_app.js

function MyApp({ Component, pageProps }) {
  return (
    <Component {...pageProps} />
  );
}

export default MyApp

安装antd

$ npm install antd --save

# or

$ yarn add antd

配置

import { ConfigProvider } from 'antd';
import zhCN from 'antd/lib/locale/zh_CN';

function MyApp({ Component, pageProps }) {
  return (
    <ConfigProvider locale={zhCN}>
      <Component {...pageProps} />
    </ConfigProvider>
  );
}

export default MyApp;
