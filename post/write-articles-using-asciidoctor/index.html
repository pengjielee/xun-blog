== 什么是Asciidoctor？

Asciidoctor 是一个 快速的 文本处理器和发布工具链，它可以将 AsciiDoc 文档转化成 HTML5、 DocBook 5 (或 4.5) 以及其他格式。–摘自官网

== Asciidoctor vs Markdown

Asciidoctor功能更强大，语法更加复杂，学习成本更高。

Markdown作为目前使用最广泛的轻量级标记语言，语法简洁。
